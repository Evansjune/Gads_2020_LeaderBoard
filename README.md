and present it in the android app.<br/>

1.Android Studio<br/>
2.Retrofit<br/>
3.API<br/>
4.Constraint Layout<br/>
5.Google Forms<br>

<b>FUNCTIONALITY</b><br/>
-Consumes data from api and present it in api 
-Submit project via Google Forms<br/>
